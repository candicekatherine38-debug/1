# Acceptance Checklist

用这份清单确认新版可以替换旧系统。

## 管理后台

- 能访问 `/admin/login` 并登录。
- 超级管理员能创建、禁用、删除普通管理员。
- 超级管理员能切换到普通管理员，并返回超级管理员。
- 管理员能修改自己的密码，修改后需要重新登录。

## 分组和号码池

- 能创建、编辑、删除分组。
- 能单个添加号码，格式为手机号 + API 地址 + 最大使用次数。
- 能批量导入 `手机号----API地址`。
- 能搜索、导出、删除、批量删除号码。
- 能启用/禁用号码，批量禁用号码。
- 能重置单个或批量重置号码使用次数。

## 链接池

- 能按分组批量生成链接。
- 能按选中的号码生成链接。
- 能导出全部链接和未访问有效链接。
- 能搜索链接、复制链接、删除链接、批量删除链接。
- 能按链接代码删除、回收链接、按链接代码禁用绑定号码。
- 链接码不能和旧数据重复。

## 公共访问

- `/link/{code}` 可打开。
- A/B/C 三种界面能按 `interface_type` 展示不同样式。
- 首次访问后记录 `first_access_time`。
- 获取验证码会绑定号码、调用对应 API，并递增使用次数。
- 超过有效期或已使用的链接不再返回有效验证码。

## 使用说明和上传

- 能给每个分组保存说明。
- 能上传图片和视频。
- 公共链接页面能展示说明和媒体。
- 未登录访问 `/upload/image`、`/upload/video` 会被拒绝。

## 定时任务

- 能保存每个分组的每日重置设置。
- 能保存删除过期号码设置。
- `php scripts/schedule.php` 能执行。
- 手动重置和手动删除过期号码能工作。

## 安全

- 所有后台 POST 请求缺少 CSRF 时会失败。
- 普通管理员不能访问其他管理员的分组、号码、链接。
- 列表数据不会因为缓存串到其他管理员。
- 数据库不开放公网访问。
- 旧系统保存过的短信 API token 已重置。
